# Player 3D criado na Godot Mobile

## O player possui:
- Movimentação com os dedos na tela
- Câmera 3D dinâmica
- Joystick
- Modelo 3D
- Animação de parado, andando, pulando e caindo
- Zoom in e Zoom out
- Efeito sonoro de pulo
- Música de fundo
---------
